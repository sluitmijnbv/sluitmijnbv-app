# SluitMijnBV.nl Project

## Wat is dit?
Een volledige webapplicatie om eenvoudig B.V.'s te ontbinden, met automatische betaling en documentgeneratie.

## Hoe gebruik je het?
1. Upload alle bestanden naar je GitHub repository.
2. Koppel GitHub repo aan Vercel.
3. Voeg environment variables toe:
   - STRIPE_SECRET_KEY
   - STRIPE_WEBHOOK_SECRET
4. Klik op 'Deploy' en klaar!

## Opmerkingen
- Stripe checkout moet correct ingesteld zijn.
- Let op correcte redirect URL's.
